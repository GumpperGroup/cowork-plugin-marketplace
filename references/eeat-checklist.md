# EEAT Bio Checklist

Use this checklist when reviewing any real estate agent bio. Every item marked ❌ is a missed opportunity for AI visibility.

## Experience
- [ ] Years licensed (specific number, not "many years")
- [ ] Total or annual transaction count
- [ ] Total or annual sales volume in dollars
- [ ] Transaction types handled (buyer, seller, both)
- [ ] Geographic scope (cities, counties, regions)
- [ ] Market conditions navigated (market shifts, booms, corrections)

## Expertise
- [ ] Professional designations spelled out with acronym (e.g., "Certified Residential Specialist (CRS)")
- [ ] Certifications listed (CNE, MRP, etc.)
- [ ] Relevant education credentials
- [ ] Niche expertise stated (luxury, first-time buyers, investment, new construction, etc.)
- [ ] Continuing education or specialized training
- [ ] Published content (blog, market reports, articles)
- [ ] Speaking or teaching credentials

## Authoritativeness
- [ ] Awards with awarding body and year
- [ ] Industry association memberships
- [ ] Leadership roles (board positions, committee chairs)
- [ ] Brokerage recognition (top producer tier, etc.)
- [ ] Media appearances or quotes
- [ ] Community involvement with named organizations
- [ ] MLS board or association involvement

## Trustworthiness
- [ ] Review rating with platform name
- [ ] Review count (specific number)
- [ ] Reviews on multiple platforms
- [ ] Brokerage affiliation (full legal name)
- [ ] Office address
- [ ] License number or verification link
- [ ] Contact information (phone + email)
- [ ] Fair housing commitment
- [ ] Client testimonial excerpt(s)

## Technical (for website bios)
- [ ] Schema.org JSON-LD markup
- [ ] Open Graph meta tags
- [ ] Meta description with agent name and location
- [ ] Page title includes agent name and city
- [ ] Headshot with descriptive alt text
- [ ] Links to review profiles
- [ ] Links to active listings or market reports
- [ ] Mobile-responsive layout
- [ ] Page loads in under 3 seconds
