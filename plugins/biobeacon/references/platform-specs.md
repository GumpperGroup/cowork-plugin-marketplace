# Platform Character Limits Quick Reference

| Platform | Section | Limit | Format | Audience |
|----------|---------|-------|--------|----------|
| Zillow | About | ~3,000 chars | Plain text | Active buyers |
| Realtor.com | About | ~4,000 chars | Plain text | Buyer/seller research |
| Google Business | Description | 750 chars | Plain text | Local searchers |
| FastExpert | Bio | ~2,000 chars | Plain text | Agent comparison |
| Homes.com | Bio | ~2,500 chars | Plain text | Early searchers |
| LinkedIn | About | 2,600 chars | Plain text + breaks | Professional network |
| LinkedIn | Headline | 220 chars | Plain text | Quick identification |
| Facebook | About | 255 chars | Plain text | Social browsers |
| Instagram | Bio | 150 chars | Plain text + emoji | Visual browsers |
| X/Twitter | Bio | 160 chars | Plain text | News/commentary |
| Personal Website | About page | No limit (250–400 words) | HTML | Deep researchers |

## Bio Version Lengths

| Version | Word Count | Use Case |
|---------|-----------|----------|
| Short | 25–40 words | Social media, quick intros, email signatures |
| Medium | 75–125 words | Platform bios, directory listings, press kits |
| Long | 150–250 words | Detailed platforms, print materials |
| Website About | 250–400 words | Personal/brokerage website |
